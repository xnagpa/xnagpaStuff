Программа может быть протестирована в командной строке или в IDE. 

Запуск программы в командной строке:
java -jar Median2.jar

Пример работы программы

c:\>java -jar Median2.jar
Enter the first array of N elements(separator is 'space'):
1 2 3 4
Enter the second array of N elements(separator is 'space'):
1 4 5 6
Median is: 3.5
